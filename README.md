# Dotfiles

Forked from [Ehren Murdick's Config](https://github.com/ehrenmurdick/config).

## Install everything:

* `rake install:all`

## Or specific stuff:

* `rake install:ack`
* `rake install:git`
* `rake install:irb`
* `rake install:vim`
* `rake install:zsh`

## Other Stuff

`setup/` contains a number of scripts that install various things when presented with a default version of Mac OS X or (later) Linux.
